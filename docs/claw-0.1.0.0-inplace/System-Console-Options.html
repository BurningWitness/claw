Description

Runtime-evaluated command-line option parsing.

Implementation

Description of the argument syntax can be found at GNU Program Argument Syntax Conventions page, which extends the short-option-only POSIX Utility Conventions.

none :: Options Identity f Source #

Empty option dictionary.

(.>) :: Options Identity f -> Option Identity help f -> Options Identity f infixl 5 Source #

Insert an option into the dictionary.

If the option has an empty list of names, nothing is added to the dictionary. Empty long names are similarly treated as no-ops. When inserting duplicate names new values take precedence.
