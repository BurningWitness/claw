Description

Options is a spine-strict tree of command-line options.

Laziness

Evaluating the root of the tree (i.e. (_ :: Options m f)) to weak head normal form evaluates the entire spine of the tree to normal form.

insert :: Option help f arg -> arg -> Options Identity f -> Options Identity f Source #

Insert an option into the tree.

If the option has an empty list of names, nothing is added to the tree. Empty long names are similarly treated as no-ops. When inserting duplicate names new values take precedence.

insertQ :: Option help f arg -> Code Q arg -> OptionsQ f -> OptionsQ f Source #

Insert an option into the tree at compilation time.

Emits warnings if any of the provided option names are inaccessible, overwrite previously added names, or contain non-portable characters. Operates identically to insert otherwise.
